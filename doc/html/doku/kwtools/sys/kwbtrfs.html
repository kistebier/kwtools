kwbtrfs

Mit diesem Script koennen Sie Btrfs-Laufwerke verwalten. Dazu gehören das erstellen, formatieren, reparieren, vergrössern, verkleinern, informieren, warten, Subvolumes und und und....

Notiz: Benoetigt root Rechte.

2. Dateisystem

1. Konfiguration

Kompressions-Algorithmus: Zur Auswahl für das komprimieren von Dateien stehen die Algorithmen "zlib" und "lzo". Der Algorithmus "zlib" ist Standard und "lzo" bietet die höhere Kompressionsrate.


2. Dateisystem

Ansicht: Hier können Sie sich Informationen über ein oder mehreren Btrfs-Laufwerken anzeigen lassen. Dazu geh&oum;ren Label, UUID, Anzahl der dazugehörigen Devices und wie diese heissen.

detaillierte Information: Hier k&oum;nnen Sie sich detaillierte Informationen über ein oder mehreren Btrfs-Laufwerken anzeigen lassen.

Blockbelegung: Hier können Sie sich die Blockbelegung von ein oder mehreren Btrfs-Laufwerken anzeigen lassen.

Defragmentierung: Hier können Sie ein oder mehrere Btrfs-Laufwerke defragmentieren. Alle Dateien werden rekursiv in den Verzeichnissen defragmentiert. Wurde in der Konfiguration ein Kompressions-Algorithmus eingestellt werden alle Dateien während der Defragmentierung damit komprimiert.

Label: Hier können Sie das Label eines Btrfs-Laufwerks ändern.

vergroessern: Hier können Sie das Btrfs-Dateisystem eines Laufwerks vergrössern.

verkleinern: Hier können Sie das Btrfs-Dateisystem eines Laufwerks verkleinern


3. Subvolumes

Ansicht: Hier können Sie sich Informationen über ein oder mehrere Subvolumes anzeigen lassen. Dazu gehören Name, UUID, Zeit der Erstellung und Subvolume ID, siehe man 8 btrfs-subvolume.

anlegen: Hier können Sie ein Subvolume anlegen.

loeschen: Hier können Sie ein Subvolume löschen. Snapshots sind auch Subvolumes.

Snapshot: Hier können Sie ein Snapshot von einem Subvolume anlegen.

Zustand Dateien: Hier können Sie sich den Zustand von Dateien, ab einer bestimmten Metadaten-Informationsnummer, anzeigen lassen.

Subvolume IDs: Hier können Sie die IDs von Laufwerken und Subvolumes ändern oder wieder herstellen.


4. Laufwerke

Ansicht: Ansicht der Speicherbelegung eines Btrfs-Laufwerks.

anlegen: Hier können Sie ein Laufwerk mit dem Btrfs-Dateisystem formatieren und ein verschlüsseltes oder unverschlüsseltes Btrfs RAID Array erstellen.

Laufwerk konvertieren: Hier können Sie ein Laufwerk mit ext2/3/4 Dateisystem nach einem Btrfs-Dateisystem konvertieren.

Laufwerk hinzufuegen: Fügt einem Btrfs-Laufwerk zusätzliche Laufwerke hinzu.

Laufwerk entfernen: Entfernt von einem Btrfs-Laufwerk ein oder mehrereLaufwerke.

Laufwerk tauschen: Tauscht von einem Btrfs-Laufwerk ein Laufwerk.


5. Gleichgewicht

Btrfs-Laufwerk: Auswahl eines Btrfs-Laufwerks auf das die einzelnen Filter angewendet werden sollen.

Datenblockgruppen: Frage, ob die Filter auf die Datenblockgruppen angewendet werden sollen (ja/nein)

Metadaten: Frage, ob die Filter auf die Metadaten Chunks angewendet werden sollen (ja/nein).

System: Frage, ob die Filter auf die System Chunks angewendet werden sollen (ja/nein).

Filter: Einstellung der einzelnen Filter die auf das Btrfs-Laufwerk angewendet werden sollen.


6. Wartung

Fehler Statistik: Anzeige der Input/Output (IO) Fehler Statistik von ein oder mehreren Btrfs-Laufwerken.

Ueberpruefung: Überprüfung der strukturellen Integrität eines Btrfs-Dateisystems.

Saeuberung: Säberung eines Btrfs-Dateisystem. Dabei werden alle Daten und Metadaten Blöcke von allen Geräten gelesen und die Prüfsummen kontrolliert. Beschädigte Blöcke werden automatisch repariert, wenn eine korrekte Kopie vorhanden ist.

Abbild: Hier können Sie ein Abbild von einem Btrfs-Dateisystem erstellen. Dabei werden nur die Metadaten des Btrfs-Dateisystems in eine Datei kopiert - die eigentlichen Daten werden jeweils als Null-Datei übernommen!

Rettung: Hier können Sie versuchen ein beschädigtes Btrfs-Dateisystem wiederherzustelllen.

Wurzel finden: Hier können Sie zufriedene Verzeichnis-Baumwurzeln finden und sie durch die Objekt ID, Generationsnummer und Level filtern.

Wiederherstellung: Hier können Sie versuchen Dateien von einem beschädigten Dateisystem zu retten und sie in einem Verzeichnis wiederherzustellen oder nur eine Liste der Subvolume Baumwurzeln anzuzeigen. Das Dateisystem-Abbild wird dabei nicht geändert.
